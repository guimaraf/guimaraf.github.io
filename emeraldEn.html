About the game

Emerald Climb was a game created for Marcos Game Dev's Gamejam Jam.

It's been a long time since the last Jam I took part in, so I decided to participate in this one with a game in which the objective is to get to the end of the mountain, taking inspiration from games like Jump Kink, Only Up, but with a precision platform game feel, yes the objective was to make a very complicated game, I'm very happy with the result achieved with the game.

Of course, if I had designed a more user-friendly game, I could have pleased more people, but this game is a study for a project I have in mind, but it won't be now that I start producing, in fact, gamejams are just for testing things.
