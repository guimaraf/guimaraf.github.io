About the game

Game created in early 2017 with friend Guilherme Vargas, for a game Jam, organized by the forum The Game Makers Brasil.
For this Jam we decided to choose a horizontal scroller shooter game, but I had never made anything in this style before. to evolve the project during the time of the Jam.

We use the Game Engine Construct 2, which is designed by Scirra.

Much of the development time was spent researching, playing similar games, noting variations in AI, boss, physics, upgrades, etc, I think today that if I were to choose a game style that I am more familiar with, I could have made a game with a longer "duration" and more "content", the time invested was very worthwhile, because in the end I had a new game in a style I never did, which was very nice and enjoyable to play.
